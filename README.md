# spectron-typescript-starter
Spectron Demo for e2e testing electron applications.

Supports:
* electron 3+ 
* Typescript 2.9.x
* Spectron 4.x (webdriverio under the hood) 

# Preconditions

## install dependencies
```bash
$ npm install
```

# Run tests

## run e2e tests with mocha reporter
```bash
$ npm run test
```
